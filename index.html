1. Definição do Sistema/System Definition
Requisitos Principais:
  • Livros: Adicionar, editar e eliminar informações de livros - ID, Título, autor, género, disponibilidade.
  • Utilizadores: Registar estudantes, professores e funcionários da biblioteca; Nome, ID, tipo de utilizador.
  • Empréstimos e retorna de livros: Realizar empréstimos e devoluções de livros - ID, livro ID, ID do usuário, data do empréstimo, data de retorno, estado.
  • Pesquisa de livros: permite pesquisar livros por nome, autor ou gênero
  • Registro histórico: Mantém um histórico completo de todos os empréstimos de livros
  • Acesso à Internet: O sistema requer conectividade à Internet
  • Site da Biblioteca: O sistema é acessado através do site da biblioteca dedicado da escola.
  • Autenticação com Email Escolar:é o email da escola
Main Requirements:
  • Books: Add, edit and delete book information - ID, Title, Author, Genre, Availability.
  • Users: Register students, teachers and library officials; Nome, ID, user type.
  • Book loans and returns: Make book loans and returns - ID, book ID, user ID, loan data, return data, status.
  • Book Search: allows you to search for books by name, author or genre
  • Historical record: Maintain a complete history of all book loans
  • Internet Access: The system requires Internet connectivity
  • Library Site: The system is accessed through the library site dedicated to the school.
  • Authentication with School Email: The user ID is the school email


Funcionalidades Básicas (CRUD)

Basic Functionalities (CRUD)

Criar:

  • Registar um novo livro
  • Registar um novo utilizador.
  • Registar um novo empréstimo.

Create:

  • Register a new book
  • Register a new user.
  • Register a new loan.

Ler:

  • Mostrar lista de livros disponíveis.
  • Mostrar informações sobre um livro específico.
  • Lista utilizadores registados.
  • Mostrar histórico de empréstimos.

Read:

  • Show list of available books.
  • Show information about a specific book.
  • List registered users.
  • Show loan history.

Atualizar:

  • Atualizar dados de um utilizador.
  • Edite informações sobre um livro.
  • Registar a devolução de um livro.

Update:

  • Update user data.
  • Edit information about a book.
  • Register the return of a book.

Eliminar:

  • Eliminar um livro da base de dados.
  • Eliminar um utilizador.

Delete:

  • Delete a book from the database.
  • Delete a user.
