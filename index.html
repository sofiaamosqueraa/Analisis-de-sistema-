1. Definição do Sistema/System Definition.
Objetivo
  • O sistema de biblioteca que desenvolvemos tem como principal objetivo agilizar e melhorar a gestão da nossa biblioteca escolar.
Goal
  • The library system we developed has the main objective of streamlining and improving the management of our school library.
Requisitos Principais:
  • Computador/telemovel
  • Utilizadores
  • Acesso à Internet: O sistema requer conectividade à Internet
  • Site da Biblioteca: O sistema é acessado através do site da biblioteca dedicado da escola.
  • Autenticação do E-mail : E-mail da escola
Main Requirements:
  • Computer/mobile phone
  • Users
  • Internet Access: The system requires Internet connectivity
  • Library Site: The system is accessed through the library site dedicated to the school.
  • Authentication Email: The school email


Funcionalidades Básicas (CRUD)

Basic Functionalities (CRUD)

Criar:

  • Registar um novo livro
  • Registar um novo utilizador.
  • Registar um novo empréstimo.

Create:

  • Register a new book
  • Register a new user.
  • Register a new loan.

Ler:

  • Mostrar lista de livros disponíveis.
  • Mostrar informações sobre um livro específico.
  • Lista utilizadores registados.
  • Mostrar histórico de empréstimos.

Read:

  • Show list of available books.
  • Show information about a specific book.
  • List registered users.
  • Show loan history.

Atualizar:

  • Atualizar dados de um utilizador.
  • Edite informações sobre um livro.
  • Registar a devolução de um livro.

Update:

  • Update user data.
  • Edit information about a book.
  • Register the return of a book.

Eliminar:

  • Eliminar um livro da base de dados.
  • Eliminar um utilizador.

Delete:

  • Delete a book from the database.
  • Delete a user.
